# hack-the-cat
Discord bot made with discord.py for capture the flag events

# Setup


**IMPORTANT!**
open your .gitignore file
add .env to the .gitignore file


Create a .env file
```
DISCORD_APPLICATION_ID = ""
DISCORD_PUBLIC_KEY = ""
DISCORD_TOKEN = ""
```

install dotenv
> python3 -m pip install python-dotenv
